# FriedMan ML Regression Project

A machine learning regression project built using Scikit-learn on the Friedman synthetic dataset. The project focuses on data preprocessing, model training, prediction analysis, and evaluation using regression metrics.

---

## Features

- Machine Learning regression model implementation
- Data preprocessing and feature handling
- Train-test data splitting
- Model evaluation using R² Score and Mean Squared Error (MSE)
- Prediction visualization using Matplotlib
- Data analysis using Pandas and NumPy

---

## Tech Stack

- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

---

## Dataset

This project uses the Friedman synthetic regression dataset provided by Scikit-learn.

The dataset is commonly used for testing regression algorithms and evaluating machine learning model performance.
